# Pokedex

## Description

Everyone loves Pokemon, and I wanted everyone to be able to customize their own Pokedex. So here is it! You can search for a pokemon and based on your favorite pokemon you can have a pokedex based on the color pallete of your favorite Pokemon!

## Tools used

+ NodeJS

+ ReactJS

+ CSS

+ SASS

+ extract-colors <https://www.npmjs.com/package/extract-colors>
